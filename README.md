# 🚀 8086Emu - Professional 8086 CPU Simulator

[![Flutter](https://img.shields.io/badge/Flutter-02569B?style=for-the-badge&logo=flutter&logoColor=white)](https://flutter.dev)
[![Assembler](https://img.shields.io/badge/8086-Assembly-green?style=for-the-badge&logo=cpu-z&logoColor=white)](https://en.wikipedia.org/wiki/Intel_8086)

**8086Emu** is a powerful and visually intuitive simulator for the Intel 8086 microprocessor. Designed for students, educators, and low-level programming enthusiasts, it brings the classic EMU 8086 experience to a modern, cross-platform environment.

---

## 🌟 Key Features | الميزات الرئيسية

- **📝 Advanced Editor**: Multi-base support (Hex, Bin, Dec), syntax-aware, and comment support.
- **⚡ Real-time Execution**: Single-step forward, **Step Back (Reverse Debugging)**, and "Run All" modes.
- **🔄 Control Flow**: Full support for **Labels**, **Jumps** (`JMP`, `JZ`, `JNZ`), and **Loops** (`LOOP`).
- **🖥️ Visual Architecture**: Detailed 16-bit register grid with H/L byte splits (AH/AL, etc.).
- **📊 Live Monitoring**: Dedicated panels for **Flags** (Manual toggling supported) and **Stack** memory (SS:SP).
- **🌍 Multi-lingual**: Full support for **English**, **Arabic**, **German**, **French**, and **Russian**.
- **🔄 Modern UI**: Professional Dark Mode with smooth animations and responsive layout.

## 🛠️ Supported Instructions | التعليمات المدعومة

| Category | Instructions |
| :--- | :--- |
| **Data Transfer** | `MOV`, `PUSH`, `POP`, `PUSHF`, `POPF` |
| **Arithmetic** | `ADD`, `ADC`, `SUB`, `SBB`, `MUL`, `IMUL`, `INC`, `DEC`, `NEG`, `CMP` |
| **Logical** | `AND`, `OR` |
| **Control Flow** | `JMP`, `JZ`/`JE`, `JNZ`/`JNE`, `LOOP`, `HLT` |

### ⚠️ Not Yet Supported | تعليمات غير مدعومة بعد
- `CALL` / `RET` (Procedures)
- `DIV` / `IDIV`
- `XOR`, `NOT`
- `TEST`
- `SHL`, `SHR`, `ROL`, `ROR`
- `INT` (Interrupts)
- Port I/O (`IN`, `OUT`)
